## Agent Contribution Mode (v0.2.0)

Agents should prefer structured contributions over direct edits.

Before modifying code:

1. Create a contribution packet in `agent_contributions/`.
2. Describe goal, files, risks, and validation plan.
3. Ensure classical baseline is included.
4. Avoid unsupported claims.
5. Request human review.

Agents must not enable automatic benchmark execution.

All contributions must remain small, reversible, and auditable.
