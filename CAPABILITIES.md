# QML Agent Lab Capabilities

This repository supports:

- Running reproducible QML vs classical benchmarks
- Comparing model performance honestly
- Serving results to AI agents via MCP interface
- Enforcing classical baseline requirements
- Providing structured experiment schemas

Limitations:

- Does not claim quantum advantage without evidence
- Does not provide production quantum infrastructure

Agent usage:

1. Read machine_index.json
2. Run benchmarks
3. Compare results
4. Propose improvements with baseline
