# Code of Conduct

## Our Pledge

We pledge to make participation in QML Agent Lab a respectful, constructive, and learning-oriented experience for everyone.

This project exists to support careful, honest, and reproducible exploration of quantum machine learning and AI-assisted development.

## Expected Behavior

Participants are expected to:

- Be respectful and constructive.
- Welcome beginners and learners.
- Critique ideas, code, and claims without attacking people.
- Document uncertainty honestly.
- Avoid hype and misleading scientific claims.
- Respect different backgrounds, skill levels, and perspectives.

## Unacceptable Behavior

Unacceptable behavior includes:

- Harassment, insults, or personal attacks.
- Discriminatory language or conduct.
- Deliberate misinformation.
- Repeatedly making unsupported scientific claims after correction.
- Sharing private information without permission.
- Introducing unsafe code or credential-handling patterns without review.

## Scientific Integrity

Because this project is research-adjacent, scientific integrity is part of community conduct.

Contributors should not misrepresent:

- toy results as real-world proof,
- simulated results as hardware results,
- single-seed results as robust evidence,
- educational examples as production systems,
- QML experiments as quantum advantage without rigorous support.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, pull requests, and other contributions that do not align with this Code of Conduct.

Maintainers may temporarily or permanently limit participation for behavior that harms the project or community.

## Reporting

To report a concern, use GitHub issues when appropriate.

If the concern is sensitive, contact the repository owner through an appropriate private channel available on GitHub.

## Scope

This Code of Conduct applies to project spaces, including GitHub issues, pull requests, discussions, documentation, and related collaboration channels.
