# Release Checklist

## Code
- MCP interface stable
- Benchmark runner validates results
- Schema matches result files

## Data
- All benchmarks include classical baseline
- All benchmarks include limitations

## Docs
- README is complete and accurate
- AGENT.md fully restored
- TASK_BOARD and NEXT_AGENT_STEP aligned

## Governance
- No auto-run workflows
- No auto-merge enabled
- PR template enforced

## Branching
- Feature branch reviewed
- PR merged to main
- Old branches cleaned up

## Final
- Tag release version
- Confirm reproducibility
