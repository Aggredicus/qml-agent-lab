# Contribution Value Rubric

This rubric helps reviewers distinguish meaningful contributions from merely valid ones.

## Scoring System

Score each category from 0 to 2.

| Dimension | 0 | 1 | 2 |
|---|---|---|---|
| Scientific Value | Trivial or unclear | Some useful signal | Clearly advances understanding |
| Reproducibility | Not reproducible | Partially reproducible | Fully reproducible |
| Baseline Integrity | Missing/weak baseline | Baseline present | Strong baseline comparison |
| Claim Discipline | Overstated | Mostly accurate | Strictly evidence-aligned |
| Maintainability | Adds complexity | Neutral | Improves clarity/system |

## Interpretation

- **0–3** → Reject or major revision
- **4–6** → Accept only if small and useful
- **7–10** → Strong contribution

## Notes

- Higher accuracy ≠ higher value
- Negative or inconclusive results can score high if well documented

## Goal

Promote signal over noise in contributions.
