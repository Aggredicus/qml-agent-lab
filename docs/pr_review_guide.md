# PR Review Guide

This guide explains how to use the Contribution Value Rubric in practice.

## Step 1 — Check correctness

Reject immediately if:

- schema is invalid
- no classical baseline
- claim is unsupported

## Step 2 — Score the contribution

Use:

`docs/contribution_value_rubric.md`

Score each dimension from 0–2.

## Step 3 — Interpret score

- 0–3 → Reject or major revision
- 4–6 → Accept only if clearly useful
- 7–10 → Strong contribution

## Step 4 — Make a decision

Consider:

- Does this improve the repo meaningfully?
- Does it add insight or just data?
- Will this help future contributors?

## Key Principle

Prefer fewer high-quality contributions over many low-value ones.
