# Prioritized Research Questions

This file defines meaningful directions for QML research within this repository.

## High Priority

- Where do quantum kernels outperform classical kernels on structured data?
- Can QML models provide benefits in low-data regimes?
- How does noise affect QML performance relative to classical baselines?

## Medium Priority

- Can hybrid classical-quantum models improve stability?
- Are there datasets where quantum feature maps improve separability?

## Low Priority

- Large-scale QML benchmarks (out of scope for this repo)
- Hardware-specific optimizations without baseline comparison

## Principle

Research questions should guide contributions toward meaningful scientific outcomes, not just higher scores.
